## Overview

Crafting a professional resume shouldn’t be a hassle. That’s why Fugoku built **Grzy**, an innovative resume builder designed to streamline the process of creating, updating, and sharing resumes. Whether you’re a student, job seeker, or professional, Grzy provides the tools to stand out in today’s competitive job market.

Grzy combines simplicity with powerful features, enabling users to customize resumes effortlessly while maintaining a polished and professional appearance.

---

## My Role

As the **Lead Developer and Founder**, I was responsible for:

- **Full-Stack Development**: Designed and implemented Grzy’s frontend using **React** and backend with **NestJS**, ensuring smooth functionality and scalability.
- **Template Design**: Created a library of 12 modern and customizable templates tailored for diverse industries and needs.
- **User Authentication**: Built secure authentication systems, including support for Google sign-in and two-factor authentication.
- **Data Management**: Integrated PostgreSQL for secure resume storage and easy tracking of resume views and downloads.

---

## Key Features

### **Customizable Templates**

Grzy offers a library of 12 professionally designed templates. Users can:

- Tailor layouts and color palettes.
- Add custom sections to highlight unique skills or achievements.
- Lock resumes to prevent unintentional edits.

### **Multi-Resume Management**

With Grzy, users can:

- Create and manage multiple resumes tailored for specific job roles.
- Track resume views and downloads to measure engagement.

### **Real-Time Sharing**

Sharing resumes has never been easier:

- Share links for online viewing or download in PDF format.
- Choose between A4 or Letter page formats for optimal printing.

### **Seamless User Experience**

Grzy is built for simplicity:

- Intuitive drag-and-drop interface.
- Light and dark theme options for a personalized experience.

---

## Technologies Used

- **React + TailwindCSS**: For a sleek, responsive user interface.
- **NestJS**: For a robust and scalable backend architecture.
- **PostgreSQL**: For secure data storage and tracking.
- **Google Chrome Integration**: Optimized for cross-browser compatibility.

---

## Challenges and Learnings

Developing Grzy provided insights into building software for diverse users:

- **Balancing Flexibility and Usability**: Grzy’s templates are fully customizable while ensuring ease of use for non-technical users.
- **Ensuring Data Security**: By implementing two-factor authentication and secure storage, Grzy keeps user data safe.
- **Streamlining Deployment**: Leveraging modern tools like Vite and containerized deployment enabled rapid iteration and scalability.
